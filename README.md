# EBAZ4205 NES (tarunes port)

This project is a port of the [tarunes](https://github.com/tomorrow56/tarunes) NES hardware implementation to the EBAZ4205 Zynq-7000 development board.

## Features
- **Video:** HDMI 480p output via TMDS adapter board
- **Audio:** I2S output (mapped to GPIO pins)
- **Input:** 5 push buttons on the adapter board mapped to A, B, SELECT, START
- **ROM Loading:** Zynq PS loads `.nes` ROMs from the microSD card into PL BRAM

## Hardware Requirements
- EBAZ4205 board with Zynq XC7Z010
- EBAZ4205 Adapter board (provides HDMI, Buttons, LEDs, microSD slot)
- microSD card formatted as FAT32

## Directory Structure
- `rtl/`
  - `tarunes_sv/` : SystemVerilog sources converted from Veryl
  - `rgb2dvi/` : TMDS encoder VHDL IP
  - `*.v` / `*.sv` : Top-level wrappers and custom modules
- `vivado/`
  - `create_project.tcl` : Vivado project generation script
  - `create_bd.tcl` : Block Design generation script
  - `constraints/` : XDC constraints
- `software/`
  - `nes_loader/` : Vitis bare-metal C application for Zynq PS

## Build Instructions

### 1. Hardware (Vivado)
1. Open Vivado 2023.2 (or later).
2. Open the Tcl Console and navigate to the `project` directory.
3. Run the project creation script:
   ```tcl
   source vivado/create_project.tcl
   ```
4. Click **Generate Bitstream**.
5. Once complete, export the hardware: **File -> Export -> Export Hardware** (include bitstream).

### 2. Software (Vitis)
1. Open Vitis IDE and set the workspace to `project/software/workspace`.
2. Create a new Platform Project using the exported `.xsa` file.
3. Modify the Board Support Package (BSP) settings to enable the `xilffs` library.
4. Create a new Empty Application Project (Standalone, C).
5. Copy `project/software/nes_loader/src/main.c` into the `src` folder of the application project.
6. Build the application.

### 3. SD Card Preparation
1. Format a microSD card as FAT32.
2. Create a folder named `nes` in the root directory.
3. Place a valid `.nes` ROM file (Mapper 0 / NROM only) inside the `nes` folder and name it `game.nes`.
   - Path should be: `D:\nes\game.nes`
4. Copy `BOOT.BIN` (generated by Vitis/Bootgen) to the root of the SD card.

## Pin Assignments
- **HDMI:** Connected to adapter board TMDS pins
- **Buttons:**
  - BTN[0] (T19) -> A
  - BTN[1] (P19) -> B
  - BTN[2] (U20) -> SELECT
  - BTN[3] (U19) -> START
  - BTN[4] (V20) -> Unused
- **I2S Audio:**
  - BCLK -> N17
  - LRCK -> R19
  - DOUT -> P20
